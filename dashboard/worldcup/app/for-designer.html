Illustrator Brief

Everything you need to deliver illustrations for the tipgame. Hand-off as WebP (preferred) or transparent PNG.

1. National-team BOBAI avatars (48 illustrations)

Concept
The existing $BOBAI character (same brain-on-BNB style as the welcome-page illustrations) wearing the football kit of each of the 48 qualified national teams. One illustration per country.
Style notes
  • Same character, same proportions across all 48 — only the kit changes.
  • Front-facing 3/4 portrait, character chest-up. Kit colour + crest visible.
  • Background: transparent (so we can place on dark dashboard cards).
  • No realistic human faces (per BOBAI brand rules).
  • Use the colours/stripes the national team is famous for. Don't reproduce trademarked club crests — abstract "BOB" emblem on the chest is fine if you want a focal element.
Specs
· Format: .webp (preferred) or .png
· Size: 512 × 512 px, square, transparent background
· Naming: avatar-<CODE>.webp using the codes below (e.g. avatar-BR.webp, avatar-ENG.webp)
· Drop into: dashboard/worldcup/app/illus/avatars/
Why it matters
These are the user avatars across the entire app — register page, dashboard profile, leaderboard rows. Currently we use flag emojis as placeholders.

2. Decorative illustrations (small, nice-to-have)

A) Pool-card flourish (Leaderboard page)
Small trophy + BOBAI illustration to anchor the "🏆 Prize Pool" card. Square, ~300 × 300 px, transparent. Place opposite the pool figure. Should communicate "growing pool" — coins, sparkles, BOBAI lifting a small trophy.
Filename: illus/pool-trophy.webp
B) Bonus-questions hero (Bonus page)
Optional decorative banner above the bonus questions list. ~1200 × 300 px, transparent, dark-mode compatible. Idea: BOBAI looking thoughtfully at a tournament bracket with a crystal ball / question mark above. Subtle, not loud.
Filename: illus/bonus-hero.webp
C) Crypto-predictions accents (Crypto page)
Tiny coin-style icons for the 3 cards (BTC, BNB, $BOBAI). 64 × 64 px each, transparent. Right now we use simple unicode/text. Filenames: illus/coin-btc.webp, illus/coin-bnb.webp, illus/coin-bobai.webp.
Style: same iconographic feel as the rest of BOBAI — playful but readable.

3. Workflow

  1. Look at the live app pages to see exactly where everything lands:
    • Register — flag grid (avatars go here)
    • Dashboard — profile card (big avatar)
    • Bonus questions — optional hero banner
    • Crypto predictions — coin icons
    • Leaderboard — pool-card flourish + avatars in rows
  2. Deliver files into dashboard/worldcup/app/illus/ following the naming above.
  3. We swap ~10 lines of HTML/CSS to point at the new files. No rework of the avatar grid needed.
